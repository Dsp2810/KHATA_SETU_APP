## 🔄 Offline Sync Flow

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                          OFFLINE SYNC FLOW                                   │
└─────────────────────────────────────────────────────────────────────────────┘

    ┌─────────────────────────────────────────────────────────────┐
    │                      USER ACTION                             │
    │             (Create transaction, update, etc.)               │
    └────────────────────────────┬────────────────────────────────┘
                                 │
                                 ▼
                    ┌────────────────────────┐
                    │   Check Network        │
                    │   Connectivity         │
                    └───────────┬────────────┘
                                │
            ┌───────────────────┴───────────────────┐
            │ OFFLINE                        ONLINE │
            ▼                                       ▼
    ┌───────────────────┐                 ┌───────────────────┐
    │ Save to Hive      │                 │ Send to API       │
    │ Local Database    │                 │ Server            │
    └─────────┬─────────┘                 └─────────┬─────────┘
              │                                     │
              ▼                                     ▼
    ┌───────────────────┐                 ┌───────────────────┐
    │ Add to Sync Queue │                 │ Update Local      │
    │ with Timestamp    │                 │ Database          │
    └─────────┬─────────┘                 └─────────┬─────────┘
              │                                     │
              │                                     ▼
              │                           ┌───────────────────┐
              │                           │     SUCCESS       │
              │                           └───────────────────┘
              │
              ▼
    ┌───────────────────────────────────────────────┐
    │           WHEN INTERNET RETURNS               │
    │                                               │
    │  1. Connectivity listener triggers            │
    │  2. Fetch pending queue                       │
    │  3. Sort by timestamp                         │
    │  4. Send each to server                       │
    │  5. Handle conflicts (server wins)            │
    │  6. Clear synced items from queue            │
    │  7. Update local with server response         │
    └───────────────────────────────────────────────┘
```
